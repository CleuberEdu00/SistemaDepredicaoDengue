# 🦟 Predição Probabilística de Dengue em Goiânia

> **Status do Projeto:** 🏗️ Em fase inicial de pesquisa, planejamento e estruturação.

## 📌 Sobre o Projeto

Este projeto tem como objetivo o desenvolvimento de um sistema computacional de suporte à decisão voltado para a vigilância epidemiológica da cidade de Goiânia (GO).

A proposta é utilizar dados históricos de casos de dengue, informações climáticas e técnicas de análise de dados para auxiliar na identificação de padrões epidemiológicos e na previsão de possíveis surtos.

O projeto está sendo desenvolvido inicialmente como trabalho acadêmico para a disciplina de Programação Orientada a Objetos 2 (POO 2), com a intenção de evoluir futuramente para um projeto mais robusto e possivelmente um Trabalho de Conclusão de Curso (TCC).

---

## 🎯 Problema

Nos últimos anos, Goiânia registrou um grande volume de casos de dengue, gerando desafios para:

* distribuição de equipes de saúde;
* planejamento preventivo;
* identificação de regiões críticas;
* gestão de recursos hospitalares;
* monitoramento epidemiológico.

Grande parte das ações ocorre apenas após o aumento significativo dos casos.

A proposta deste projeto é estudar formas de utilizar tecnologia e análise de dados para apoiar decisões preventivas.

---

## 💡 Objetivo

Desenvolver um sistema capaz de:

* analisar dados epidemiológicos;
* identificar padrões de crescimento de casos;
* integrar dados climáticos;
* gerar previsões probabilísticas;
* auxiliar visualmente a tomada de decisão.
